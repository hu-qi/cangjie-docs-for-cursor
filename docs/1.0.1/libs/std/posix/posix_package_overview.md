
# std.posix

## 功能介绍

posix 包封装 POSIX 系统调用，提供跨平台的系统操作接口。

本包提供多平台统一操控能力，目前支持 `Linux` 平台，`macOS` 平台，`Windows` 平台与 `HarmonyOS` 平台。

> **注意：**
> 
> 未来版本即将废弃本包全部内容。
